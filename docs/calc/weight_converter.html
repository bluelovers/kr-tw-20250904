不同單位的價格換算 / 서로 다른 단위의 가격 환산

公斤(KG) / 킬로그램(KG) 元(원)
公克(g) / 그램(g) 元(원)
台斤 / 대근 元(원)
台兩 / 냥 元(원)
填入任一價格，不同單位會自動轉換
아무 가격이나 입력하면, 다른 단위가 자동으로 변환됩니다

換算不同重量單位下的價格，包括公斤，公克，台斤，台兩。例如 1公斤130元，等於1台斤多少錢？ 1台斤28元，換算成1公斤是多少錢？
서로 다른 중량 단위의 가격을 환산합니다. 포함: 킬로그램, 그램, 대근, 냥. 예시: 1킬로그램이 130원이라면, 1대근은 얼마인가요? 1대근이 28원이라면, 1킬로그램은 얼마인가요?

如何使用這個價格計算工具 / 이 가격 계산 도구 사용 방법

  • 第一，先確定重量單位是你要的，可以調整成300克，0.5斤(半斤)或2兩。
  • 各個重量單位確定好，再輸入你知道的價格。
  • 其他價格會自動計算
  • 먼저, 원하는 중량 단위를 확인하세요. 예: 300그램, 0.5근(반근), 2냥.
  • 단위를 확정한 후, 알고 있는 가격을 입력하세요.
  • 나머지 가격은 자동으로 계산됩니다

重量單位轉換 / 중량 단위 변환

  • 1台斤等於600公克 / 1대근 = 600그램
  • 1台斤是16兩，半斤8兩 / 1대근 = 16냥, 반근 = 8냥
  • 1兩是37.5公克 / 1냥 = 37.5그램
  • 1公斤 = 1000 公克 / 1킬로그램 = 1000그램

台斤換算公斤價格 / 대근을 킬로그램 가격으로 환산

1台斤30元，1公斤是多少錢？
1台斤=600克=0.6公斤。
30 ÷ 1 ÷ 0.6 = 50
答：1公斤是50元

1대근이 30원일 때, 1킬로그램은 얼마인가요?
1대근 = 600그램 = 0.6킬로그램
30 ÷ 1 ÷ 0.6 = 50
답: 1킬로그램은 50원

公斤換算台斤價格 / 킬로그램을 대근 가격으로 환산

1公斤120元，1台斤是多少錢？
1公斤=1000克，1台斤=600公克。
120 ÷ 1000 × 600 = 72
答：1台斤是72元

1킬로그램이 120원일 때, 1대근은 얼마인가요?
1킬로그램 = 1000그램, 1대근 = 600그램
120 ÷ 1000 × 600 = 72
답: 1대근은 72원
